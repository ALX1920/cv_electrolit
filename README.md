# CV Electrolit

Repositorio destinado al desarrollo y despliegue del CV profesional de Alejandro Martínez. El proyecto utiliza una estructura modular en CSS y JavaScript para facilitar mantenimiento y futuras mejoras.

## Tecnologías utilizadas
- HTML5
- CSS modular
- JavaScript vanilla

##  Despliegue
El sitio se publica mediante GitHub Pages desde la rama principal.

##  Notas
Este repositorio contiene únicamente los archivos finales del CV, sin exponer código innecesario ni configuraciones internas.
